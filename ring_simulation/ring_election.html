*To crash a node, click on it. To reactivate, click on it again.

How to Use the Simulator:

Enter the structure of the Ring. This is done by entering between 3 and 8 distinct and non-negative numbers.
Next, select a speed for the simulation to run at. If 'Step' is selected, press the 'Step' button to continue the simulation.
To reset the simulator, press the 'Reset' button next to the 'Start' button.
